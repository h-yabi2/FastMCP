## 仮想環境の作成とアクティベート

本プロジェクトでは、仮想環境を作成し、その中で作業を行うことを推奨します。  
仮想環境を利用することで、他のプロジェクトと依存関係が混ざることを防げます。

### 仮想環境の作成

```bash
python -m venv .venv
```

### 仮想環境のアクティベート

- **Mac/Linux:**
  ```bash
  source .venv/bin/activate
  ```
- **Windows:**
  ```cmd
  .venv\Scripts\activate
  ```

仮想環境をアクティベートした状態で、依存パッケージのインストールやコマンドの実行を行ってください。
